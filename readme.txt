Sends personalized SMS to customers on each new order as well as order status change. You can also receive SMS notifications on new orders.

== Description ==

Thinking of sending personalized text messages to your customers on each new order? Feel free to use our best plugin.

We are planning to add more SMS gateways as this plugin matures. Please share your feedback so we can prioritize based on inputs.

You can use this plugin to Send SMS on:
* Each new order
* Order status change
* Custom SMS to your specified numbers
* Admin notification for new each new order
* Personalized templates for all SMS's
* Dynamic variable options to choose from like, customer's first name, last name etc.

Current SMS Gateway Provider supported:
* Twilio
* Plivo
* Burst SMS
* Nexmo
* VoodooSMS

We are always open for feedback, so please feel free to share your [feedback](https://www.codebite.pk) about this plugin.

Our plugin is specially made to work with WooCommerce. For more information please visit [WooCommerce.com](https://woocommerce.com/).

== Installation ==

1. Install plugin from WordPress Plugins.
2. Activate our plugin from your plugins page.
3. Woo SMS Pro will appear in the sidebar. Click it.
4. Go to SMS Gateway tab and choose your SMS Gateway provider and set its configuration options.
5. Go to SMS Templates tab and set your personalized SMS templates.
6. Go to SMS Sender tab and try sending a test SMS to your number to see if plugin works OK.
7. If all is set, then go to General tab and activate your plugin and choose your options.

== Frequently Asked Questions ==

= Is the plugin free? =

Yes, it is free. But the gateway you will be using to send SMS, will charge you for each SMS you send.

= Can you add my SMS gateway in your plugin? =

Sure, we are always open for feedback.
